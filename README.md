#  CI/CD Pipeline using Jenkins & Docker

##  Project Overview
This project demonstrates a complete CI/CD pipeline using Jenkins and Docker.

##  Tools Used
- Git
- GitHub
- Jenkins
- Docker

##  Workflow
1. Code pushed to GitHub
2. Jenkins pulls the code
3. Docker builds image
4. Container runs application
5. App accessible in browser

##  Output
http://localhost:8081

##  Files
- index.html → Web page
- Dockerfile → Container setup
- Jenkins Pipeline → Automation

##  Features
- Automated build and deployment
- Docker containerization
- CI/CD pipeline integration
